# Title: Topsis For Text Classification

# 1. Methodology

Text Classification ----> Collecting prediction results from different classification models ----> Creating performance dataset using evaluation metrics ----> Applying Topsis and Generating Topsis Value ----> Comparing different models using graphs

# 2. Description

- Dataset: self-generated labeled dataset
- Metrics Used: Accuracy, F1 Score, Inference Time
- Weights: Non-negative (> 0)
- Impacts: Positive/Negative (+/-)
    Accuracy        : +
    F1 Score        : +
    Inference Time  : -
- Output: Table & Graph

# 3. Objectives

- Applying basic mathematics to find the TOPSIS score.
- Finding the best model for text classification.
- Comparing models based on multiple evaluation criteria.
- Changing weights and impacts to observe differences in ranking.
- Understanding multi-criteria decision-making in NLP.
